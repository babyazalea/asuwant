# asuwant-frontend
> 뉴스를 골라볼 수 있는 웹 앱 asuwant의 Front-end

asuwant는 각국 뉴스를 골라볼 수 있는 웹 앱입니다. asuwant의 프론트엔드는 CRA를 통해 생성된 React 프로젝트를 기반으로 합니다.
